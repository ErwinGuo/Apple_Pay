# Apple_Pay
Applepay实现demo

简单的Apple pay的集成实现：

主要步骤：

1.一个有效的苹果开发者账号

2.从苹果开发者中心（https://developer.apple.com）创建APPID、创建对应的Applepay证书

3.project -> target -> Capabilities -> Apple Pay 设为on

4.导入框架 -> import<PassKit/PassKit.h>

5.开始集成
